Implement a file-based key-value store in Python with the following features: get/set/delete operations, TTL (time-to-live) support for keys that auto-expire, LRU eviction when the store exceeds a configurable max number of entries, thread-safe concurrent access using locks, persistence to a JSON file on disk. Provide a clean API class and comprehensive tests covering TTL expiry, LRU eviction, and concurrent access scenarios.
